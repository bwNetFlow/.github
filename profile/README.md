# bwNetFlow

Das Projekt bwNetFlow befasste sich mit der Verkehrsanalyse im BelWü Netzwerk.
Das Projektziel ist die Realisierung einer Plattform zur Auswertung des Datenaufkommens in BelWü.
Projektlaufzeit war 01.12.2017 bis 30.06.2020.

# Weiterentwicklung der entstandenen Software 
Die in bwNetFlow erstellte Software wurde im Rahmen der nachfolgenden [bwNET Projekten](https://bwnet2.belwue.de/) zu der flowpipeline Software weiterentwickelt.
Diese wird aktuell von mehreren Organisationen produktiv eingesetzt und auf https://codeberg.org/BelWue/flowpipeline aktiv weiterentwickelt.
